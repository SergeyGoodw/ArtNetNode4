 
# ArtNetNode 4

Small Xmega32 and enc28j60 based DYI ArtNet Node with 4 DMX ports.

## Features:
    - 4 DMX Outputs(full 512 channels)
    - Testes framerate: 4x 52 FPS
    - Web Interface for configuration(WIP, see TODO)
    - industry standard 19" 1U housing
